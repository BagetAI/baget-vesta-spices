# Vesta Spices Landing Page

A high-converting "Coming Soon" page for Vesta Spices, built with Next.js 14, Tailwind CSS, and Baget Database integration.

## Stack
- Framework: Next.js 14
- Styling: Tailwind CSS
- Icons: Lucide React
- Database: Baget Public API
- Theme: Tech-Forward Culinary

## Deployment
Deployed via Vercel.
